# Security Policy

## Scope
Quadrama is a live end-to-end encrypted messaging service with real users.

This repository contains documentation, protocol notes, and threat modeling
only. It does NOT contain production infrastructure or full source code.

## Authorized Testing
No testing, probing, scanning, or attacking of live systems, servers,
networks, or user accounts is authorized by this repository.

Any security testing against production systems requires explicit
written permission.

## Reporting Vulnerabilities
If you believe you have found a security issue related to Quadrama,
please report it responsibly:

- Email: security@quadrama.ch
- Please do NOT open public GitHub issues for sensitive vulnerabilities.

## Legal
Unauthorized testing of live systems may be illegal.
Nothing in this repository grants permission to bypass safeguards
or attempt exploitation.
