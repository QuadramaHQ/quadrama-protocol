# Quadrama

**Private. Secure. Simple.**

Quadrama is a free, open-source messenger for private 1:1 conversations.
No account. No phone number. No data collection.

Try it at **[quadrama.ch](https://quadrama.ch)**

---

## Why Quadrama?

Most messaging apps collect your data, require an account, or store your messages on their servers.

Quadrama works differently:

- **No account required** — open the browser and start chatting
- **End-to-end encrypted** — only you and your contact can read your messages
- **No message storage** — messages are never stored on the server
- **No logs** — the server does not log connections or activity
- **Open source** — anyone can review the code

---

## How to Use

1. Open [quadrama.ch](https://quadrama.ch)
2. Click **Load/create Identity**
3. Click **Connect**
4. Create a room or enter a room code shared by your contact
5. Both join the same room
6. Compare the **Safety Code** with your contact (by phone, in person, or another trusted channel)
7. Click **Verify**
8. Start chatting securely

---

## Security

Quadrama uses the **Double Ratchet** protocol — the same cryptographic foundation used by Signal — to provide:

- **Forward Secrecy** — past messages remain secure even if keys are later compromised
- **Post-Compromise Security** — future messages are secure even after a key compromise
- **Identity verification** — the Safety Code ensures you are talking to the right person

All cryptography runs in your browser. The server never has access to your messages or keys.
